## C++ Game Engine

### Build:

```sh
    mkdir build
    cd build
    cmake ..
```

### Linux:

```sh
    make -j4
```

### Windows:

Open the generated solution in Visual Studio (make sure you use the latest version) and build the entire solution. Make sure you set "engine" as Startup Project.

External dependencies:
* GLEW-2.1.0
* GLFW-3.3.2
* Assimp-5.0.0
